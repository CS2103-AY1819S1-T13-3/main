== Overview

JxMusic Player is a desktop music player application developed during a software engineering module. The application was built upon https://se-edu.github.io/addressbook-level4/[AddressBook Level-4]. The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.
